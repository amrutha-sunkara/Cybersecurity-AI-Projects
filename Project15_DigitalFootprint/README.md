# Project 15 - Digital Footprint Mapping Generator

## Objective

This project helps users identify their digital footprint and assess privacy risks associated with the information they share online.

## Features

- Platform selection
- Personal data analysis
- Risk score generation
- Privacy recommendations
- Overall security assessment

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open:

```text
index.html
```

in a browser or use Live Server.

## Output

The application generates:

- Platform-wise risk assessment
- Privacy exposure analysis
- Recommendations for improving digital privacy
